Making the Fundamental Layout Responsive

The Fundamental Layout challenge is a task given at the end of the Mozilla Developer Network's layout module. It tests comprehension of layout skills using a full range of use cases for different CSS methods. The layout implemented is interesting in its own right in that it uses many key modern front-end development techniques such as semantic HTML, CSS Grid, and flexbox. One drawback, though, is that the end result is not a responsive website, so when viewed on a mobile device it looks jumbled.

To fix these issues I added a few simple CSS tweeks using @media queries to make the layout responsive. Tweeking the article photo starts by adding the border-radius property set to a value of 50%. Adding in a @media query at 480px to change the display from grid to block means that it can now be viewed on mobile screens.
